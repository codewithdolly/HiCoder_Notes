9. Explain call stack in javascript? *
    -A call stack is a mechanism for an interpreter (like the JavaScript interpreter in a web browser) to keep track of its place in a script that calls multiple functions. basically call stack works as LIFO. LIFO- Last-In First-Out. Here we can write many function inside a function and invoke it.

    -pre increment means Before assigning the value to the variable we write ++ ( ex: ++10 ). Its add +1 before executing the value.In Post Increment - It assign ++Before assigning the value to the variable (ex: 10++). Its add +1 after executing the value.


11. What is key logger? *
    -Keystroke logging, often referred to as keylogging or keyboard capturing, is the action of recording the keys struck on a keyboard, typically covertly, so that a person using the keyboard is unaware that their actions are being monitored. Data can then be retrieved by the person operating the logging program.
